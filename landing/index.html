FAQ

What is Google Flow?

Google Flow is Google's AI-powered image and video generation tool available at labs.google. FlowAuto automates the generation process so you can work at scale.

Does FlowAuto send my data to external servers?

All your data (prompts, settings, images) stays locally in your browser. The only external calls are license validation, optional AI API requests (when you configure your own key or use Pro+ proxy), and optional notification delivery to Telegram/Discord.

What is Chain Mode?

Chain Mode takes the output of each generation and feeds it as the reference image for the next task. This lets you build iterative creative workflows where each result builds on the previous one.

Can I use my own OpenAI or Gemini API key?

Yes. Pro users can configure their own API keys for prompt enhancement and variant generation. Pro+ users can also use our proxy without needing their own key.

Is there a refund policy?

Pro licenses are one-time purchases. If you're not satisfied, contact us within 14 days for a full refund. Pro+ subscriptions can be cancelled any time.
