# FlowAuto — Google Flow 批量自动化工具

在 Google Flow (labs.google) 上自动化批量图像和视频生成。通过智能队列管理，规模化创建专业内容。

## 核心功能

- **批量队列** — 排列多个提示词，依次生成并自动下载
- **链式模式** — 上一次生成的结果自动成为下一个任务的参考图 (Pro)
- **隐身模式** — 模拟人工操作节奏，自然运行 (Pro+)
- **项目管理** — 将工作组织到多个独立项目中
- **AI 提示词引擎** — 使用 OpenAI/Gemini 增强提示词并生成创意变体
- **通知集成** — 批量完成时通过 Telegram 或 Discord 通知 (Pro+)
- **智能下载** — 按项目文件夹自动分组下载 (720p-4K)
- **M x N 组合** — 多参考图 x 多提示词批量组合 (Pro)

## 使用方式

1. 打开任意 Google Flow 项目
2. 点击 FlowAuto 侧边栏
3. 输入提示词（每行一个或从文件导入）
4. 点击「开始生成」— FlowAuto 自动完成后续操作
