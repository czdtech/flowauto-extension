# FlowAuto — Batch Automation for Google Flow

Automate batch image and video generation on Google Flow (labs.google). Create production-ready content at scale with intelligent queue management.

## Key Features

- **Batch Queue** — Queue multiple prompts and generate them sequentially with auto-download
- **Chain Mode** — Each generation's output becomes the next task's reference image (Pro)
- **Stealth Mode** — Human-like timing randomization for natural operation (Pro+)
- **Project Management** — Organize work into multiple independent projects
- **AI Prompt Engine** — Enhance prompts and generate creative variants with OpenAI/Gemini
- **Notifications** — Get notified via Telegram or Discord when batches complete (Pro+)
- **Smart Download** — Auto-download with project folder grouping (720p-4K)
- **M x N Combinations** — Combine multiple reference images with multiple prompts (Pro)

## How It Works

1. Open any Google Flow project
2. Click the FlowAuto side panel
3. Enter your prompts (one per line or import from file)
4. Click "Start" — FlowAuto handles the rest
